ヘルプ - 使い方とショートカット

基本操作

  • 画像を開く：「画像を開く...」ボタン、またはウィンドウ内へドラッグ&ドロップ。複数ファイルまとめて開けます。
  • ファイル切替：左のファイル一覧をクリック、「◀前」「次▶」ボタン、または割り当てキー。
  • ファイル一覧の ✔ は保存済みのしるしです。
  • 画像を切り替えるとモザイクはリセットされます（未保存なら確認ダイアログが出ます）。

パターンを付ける

  • ペン：ドラッグした箇所にリアルタイムでパターンを適用。ペンサイズを変更できます。
  • 四角 / 円：ドラッグで範囲を選択してパターンを適用。Shiftを押しながらで正方形／正円になります。
  • 黒線 / 白線：クリックした点が始点、ドラッグ中はリアルタイムでプレビュー、離した点が終点になります。端点は□（角型）です。線の太さは粒度スライダーで調整できます。
  • 消しゴム：ドラッグした箇所のパターンを元画像に戻します。カーソルが赤いリングで表示されます。ペンと同じ感覚でなぞって使えます。
  • パターン：モザイク・ぼかし・ノイズから選択。黒線・白線はパターン選択に関わらず直接描画されます。操作ごとに記録されるため、1枚の画像に異なるパターンを混在させることができます。
  • 粒度：パターンの粗さ・線の太さ（2〜64px）。操作ごとに固定なので、後から変えても既存の操作には影響しません。
  • ペンサイズ：4〜200px。ペン・消しゴム選択時はカーソルにサイズのリングが出ます。

編集

  • 元に戻す：直前の操作を取り消します。全消去も戻せます（履歴は最大200件）。
  • 全消去：全てのモザイクを削除。

保存

  • PNG形式で保存します。透明部分はそのまま保持されます。
  • ファイル名：元の名前の文頭または末尾に文字を追加（デフォルト末尾 _mz）。ツールバーに保存名プレビューが出ます。
  • 保存して次へ：保存後に自動で次のファイルへ移動。連続作業向け。

表示

  • 拡大 / 縮小：＋／−ボタン、または割り当てキー。10〜800%までズーム。
  • 適合：真ん中のパーセント表示ボタンをクリックすると、ウィンドウにフィットするサイズに戻ります。
  • 拡大時はステージ内をスクロールできます。画像を切り替えると自動で「適合」に戻ります。

擬音スタンプ（右パネル）

  • 作る：書体・文字（改行OK）・縦書き・ボールド・文字色・縁取り色/太さ・不透明度を設定し、プレビューで確認できます。
  • 配置：「➕ 中央に配置」ボタン、または割り当てキーでカーソル位置に配置。「よく使うスタンプ」のマスをクリックすると、その内容ですぐ配置されます。
  • 自分の画像をはんこに：「📁 画像を追加」ボタン、またはキャンバスへ直接画像ファイルをドラッグ&ドロップすると、その場にはんことして配置されます（複数可）。
  • 選択・編集：キャンバス上のスタンプをクリックすると選択され、右パネルの内容がそのスタンプの編集フォームになります。文字や色を変えるとリアルタイムに反映されます。
  • 移動：スタンプ本体をドラッグ。Shift+クリックで複数選択でき、まとめて移動・削除できます。
  • 変形：四隅の黄色いハンドルをドラッグすると自由変形（台形にも可）。Shiftを押しながらドラッグすると、対角を固定したまま形状を保った拡大・縮小になります。
  • 回転：緑色のハンドルをドラッグ。
  • 複製 / 最前面へ / 変形リセット / 選択削除：右パネルのボタンから操作できます。
  • よく使うスタンプに保存：⭐ボタンで現在の設定（選択中なら変形も含む）を登録。テキストの擬音だけでなく、自分で追加した画像のはんこも選択して保存できます。次回からワンクリックで同じ内容・同じ変形を配置できます。登録した項目はマス上で✕ボタンから削除できます。
  • タブ分け：「よく使うスタンプ」の上にある番号タブで登録先を切り替えられます。「+」で新しいタブを追加できます。
  • 書き出し / 読み込み：登録した「よく使うスタンプ」をJSONファイルとして書き出し・読み込みできます。読み込み時は既存の登録を消さず、末尾に追加されます。
  • PNG保存時は、モザイク編集済みの画像にスタンプも合成されて1枚のPNGになります（見た目そのままの位置・変形で書き出されます）。

ショートカット（固定）

Ctrl / ⌘ + Z
元に戻す
Ctrl / ⌘ + S
保存
Delete / Backspace
スタンプ選択中はそのスタンプを削除／未選択時は選択中の画像をリストから削除（確認なし）
Esc
ドラッグ中の範囲選択・ペン操作をキャンセル
Shift + ドラッグ（キャンバス空白部）
画像をスクロール（移動）。拡大表示時に使用
Shift（四角／円ツール）
ドラッグ中：正方形・正円に固定
Shift（スタンプの角ハンドル）
ドラッグ中：対角を固定して形状を保った拡大・縮小
Shift + クリック（スタンプ）
複数選択に追加／解除

ショートカット（設定可能）

「キー設定...」ボタンから変更できます。設定はブラウザに保存され、次回起動時も維持されます。

Tips

  • 複数ファイルをまとめて開けば、キーで送り／戻ししながら次々作業できます。
  • 「保存して次へ」を割り当てキー未対応の作業でも、PNG保存＋次へ送りのキーを続けて押せば同じ動きにできます。
  • 粒度を粗くする→広範囲を大胆に、細かくする→ピンポイントに、という使い分けができます。
  • モザイクを描いたあとに続けてスタンプを置きたいときは、割り当てキー（デフォルト6）でカーソル位置にサッと配置し、あとから角のハンドルで調整するとスムーズです。
  • 画像を切り替えるとスタンプもリセットされます。よく使う擬音やはんこ画像は事前に「よく使うスタンプに保存」しておくと、別画像でもすぐ呼び出せます。
