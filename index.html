ヘルプ - 使い方とショートカット

基本操作

  • 画像を開く：「画像を開く...」ボタン、またはウィンドウ内へドラッグ&ドロップ。複数ファイルまとめて開けます。
  • ファイル切替：左のファイル一覧をクリック、「◀前」「次▶」ボタン、または割り当てキー。
  • ファイル一覧の ✔ は保存済みのしるしです。
  • 画像を切り替えるとモザイクはリセットされます（未保存なら確認ダイアログが出ます）。

モザイクを付ける

  • ペン：ドラッグしたところにリアルタイムでモザイク。ペンサイズを変更できます。
  • 四角 / 円：ドラッグで範囲を選択してモザイクを適用。Shiftを押しながらで正方形／正円になります。
  • 粒度：モザイクの粗さ（2〜64px）。操作ごとに固定なので、後から粒度を変えても既存モザイクは変わりません。
  • ペンサイズ：4〜200px。ペン選択時はカーソルにサイズのリングが出ます。

編集

  • 元に戻す：直前の操作を取り消します。全消去も戻せます（履歴は最大200件）。
  • 全消去：全てのモザイクを削除。

保存

  • PNG形式で保存します。透明部分はそのまま保持されます。
  • ファイル名：元の名前の文頭または末尾に文字を追加（デフォルト末尾 _mz）。ツールバーに保存名プレビューが出ます。
  • 保存して次へ：保存後に自動で次のファイルへ移動。連続作業向け。

表示

  • 拡大 / 縮小：＋／−ボタン、または割り当てキー。10〜800%までズーム。
  • 適合：真ん中のパーセント表示ボタンをクリックすると、ウィンドウにフィットするサイズに戻ります。
  • 拡大時はステージ内をスクロールできます。画像を切り替えると自動で「適合」に戻ります。

ショートカット（固定）

Ctrl / ⌘ + Z
元に戻す
Ctrl / ⌘ + S
保存
Delete / Backspace
選択中の画像をリストから削除（確認なし）
Esc
ドラッグ中の範囲選択・ペン操作をキャンセル
Shift
四角ツールで正方形、円ツールで正円に固定

ショートカット（設定可能）

「キー設定...」ボタンから変更できます。設定はブラウザに保存され、次回起動時も維持されます。

Tips

  • 複数ファイルをまとめて開けば、キーで送り／戻ししながら次々作業できます。
  • 「保存して次へ」を割り当てキー未対応の作業でも、PNG保存＋次へ送りのキーを続けて押せば同じ動きにできます。
  • 粒度を粗くする→広範囲を大胆に、細かくする→ピンポイントに、という使い分けができます。
